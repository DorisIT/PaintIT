## The Idea
The project aims to create a desktop version of the popular mobile game Draw Something developed by OMGPop. The game is simple enough, one player draws a given word and the second player guesses what’s depicted.


## General Characteristics
* The game is turn based.
* There are 2 players.
* The players take turns drawing and guessing
* The goal is to guess as many correct words in a row as possible.
* There are no time constraints.
* The player that draws is given a word to depict.
