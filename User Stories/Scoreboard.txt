# User stories

Story Identifier: STK001
Story Name: Scoreboard


## Description 

As a competitive player, I want to be able to track my score during gameplay

## Confirmation
Some kind of counter keeping track of streaks of correct consecutive guesses.

### Functional


* Can I keep track of score while the game progresses?
* Can I save my score between sessions?

### Non-functional

- availability
* Can I view the score at a later time (after a session is done)?
* Can I compare my score to others?

- security...
