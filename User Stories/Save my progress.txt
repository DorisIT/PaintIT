# User story 

Story Identifier: STKXXX
Story Name: Save my Progress

## Description 

As a player in the game, I need to be able to save my progress when i exit the game so I can continue another time.


## Confirmation

- Is the progress saved when I exit the game?
- Is it clear to me that the progress is saved when I exit the game?
- Can I choose to discard my progress?
