# User stories

Story Identifier: STK001
Story Name: Instructions        


## Description 

As a player in this game, I want to be able to get instructions/rules for the game so that I know how the game is played

## Confirmation
When it is clear to the players how to play the game.

### Functional


* Are sufficient instructions given to the players?

### Non-functional

- availability
* Are/should the players able to access the instructions during the game?

- security...
   * Is it impossible to misinterpret the rules?
